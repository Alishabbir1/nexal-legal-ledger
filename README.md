# Nexal Legal — Client Ledger & Cashbook (Browser Version)

## 🚀 Quick Start - No Installation Required!

Simply open `index.html` in your web browser. That's it!

### How to Run:

1. **Double-click `index.html`** - It will open in your default browser
2. **Or right-click `index.html`** → Open With → Choose your browser

No Python, no server, no installation needed!

## Features

✅ **Client Ledger Management** - Create clients and track transactions
✅ **Cashbook** - Manage client money with cheque clearance handling
✅ **SRA Compliance** - Deficit prevention built-in
✅ **Export to CSV** - Download reports for Excel
✅ **All data stored locally** - Your data stays on your computer

## How It Works

- All data is stored in your browser's localStorage
- No internet connection required
- Data persists between sessions
- Completely private and secure

## Browser Compatibility

Works in all modern browsers:
- Chrome/Edge
- Firefox
- Safari
- Opera

## Data Storage

Your data is saved automatically in your browser. To backup:
1. Go to Reports page
2. Click "Export Ledger to CSV" and "Export Cashbook to CSV"
3. Save the CSV files

## Notes

- This is a client-side only application
- Data is stored in browser localStorage
- Clear browser data will delete your records (export first!)
- For production use, consider the server version for better data security
